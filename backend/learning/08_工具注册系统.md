### 8.2.2 ParameterDefinition 参数定义

```python
@dataclass
class ParameterDefinition:
    """工具参数定义"""
    name: str                    # 参数名
    type: str                    # 参数类型
    description: str             # 参数描述
    required: bool = True        # 是否必填
    default: Any = None          # 默认值
    enum: List[Any] = None      # 可选值列表
```

**工程化实践**：参数验证策略

```python
def _validate_arguments(
    self,
    tool: ToolDefinition,
    arguments: Dict[str, Any]
) -> Dict[str, Any]:
    """验证并准备参数"""
    validated = {}
    
    for param in tool.parameters:
        if param.name in arguments:
            # 参数存在，使用传入值
            validated[param.name] = arguments[param.name]
        elif param.required:
            # 必填参数缺失，抛出错误
            raise ValueError(f"Required parameter '{param.name}' missing")
        elif param.default is not None:
            # 使用默认值
            validated[param.name] = param.default
    
    return validated
```

### 8.2.3 ToolDefinition 工具定义

**关键功能**：OpenAI格式转换

```python
def to_openai_format(self) -> Dict[str, Any]:
    """转换为OpenAI函数调用格式"""
    properties = {}
    required = []
    
    for param in self.parameters:
        properties[param.name] = param.to_dict()
        if param.required:
            required.append(param.name)
    
    return {
        "type": "function",
        "function": {
            "name": self.name,
            "description": self.description,
            "parameters": {
                "type": "object",
                "properties": properties,
                "required": required
            }
        }
    }
```

## 8.3 ToolRegistry 工具注册表

### 8.3.1 注册表功能

**工程化设计**：单例模式与集中管理

```python
class ToolRegistry:
    """
    工具注册表
    
    功能：
    - 工具注册与发现
    - 按类型查找工具
    - 工具元数据管理
    """
    
    def __init__(self):
        self._tools: Dict[str, ToolDefinition] = {}
        self._tools_by_type: Dict[ToolType, List[ToolDefinition]] = {}
```

### 8.3.2 核心方法

```python
# 注册工具
def register(self, tool: ToolDefinition) -> None:
    self._tools[tool.name] = tool
    if tool.tool_type not in self._tools_by_type:
        self._tools_by_type[tool.tool_type] = []
    self._tools_by_type[tool.tool_type].append(tool)

# 获取工具
def get(self, name: str) -> Optional[ToolDefinition]:
    return self._tools.get(name)

# 列出工具（支持过滤）
def list_tools(self, tool_type: Optional[ToolType] = None) -> List[ToolDefinition]:
    if tool_type:
        return self._tools_by_type.get(tool_type, [])
    return list(self._tools.values())

# 获取OpenAI格式的工具定义
def get_openai_format_tools(self, tool_type: Optional[ToolType] = None):
    tools = self.list_tools(tool_type)
    return [t.to_openai_format() for t in tools]
```

## 8.4 ToolExecutor 工具执行器

### 8.4.1 执行流程

**产品化思维**：完整的错误处理与用户反馈

```python
async def execute(
    self,
    tool_name: str,
    arguments: Dict[str, Any],
    enterprise_id: Optional[str] = None,
    user_id: Optional[str] = None
) -> ToolExecutionResult:
    """
    执行工具
    
    执行流程：
    1. 获取工具定义
    2. 验证企业上下文
    3. 验证参数
    4. 添加上下文参数
    5. 执行工具
    6. 返回结果
    """
    import time
    start_time = time.time()
    
    try:
        # 1. 获取工具定义
        tool = self.registry.get(tool_name)
        if not tool:
            return ToolExecutionResult(
                success=False,
                error_message=f"Tool '{tool_name}' not found"
            )
        
        # 2. 验证企业范围（产品化：数据隔离）
        if tool.enterprise_scoped and not enterprise_id:
            return ToolExecutionResult(
                success=False,
                error_message="Enterprise context required"
            )
        
        # 3. 验证参数（工程化：输入验证）
        validated_args = self._validate_arguments(tool, arguments)
        
        # 4. 添加上下文参数
        if tool.enterprise_scoped:
            validated_args["enterprise_id"] = enterprise_id
        if tool.requires_auth:
            validated_args["user_id"] = user_id
        
        # 5. 执行处理函数（支持同步/异步）
        if inspect.iscoroutinefunction(tool.handler):
            result = await tool.handler(**validated_args)
        else:
            result = tool.handler(**validated_args)
        
        execution_time = (time.time() - start_time) * 1000
        
        return ToolExecutionResult(
            success=True,
            data=result,
            execution_time_ms=execution_time
        )
        
    except ValidationError as e:
        return ToolExecutionResult(
            success=False,
            error_message=f"Parameter validation failed: {str(e)}"
        )
    except Exception as e:
        return ToolExecutionResult(
            success=False,
            error_message=f"Tool execution failed: {str(e)}"
        )
```

```python
    """工具执行结果"""
    success: bool
    data: Any = None
    error_message: str = None
    execution_time_ms: float = 0
```

## 8.6 工具自动发现与加载

### 8.6.1 Loader 设计

**工程化思维**：自动发现，减少手动配置

```python
def discover_tools(module_path: str, package_path: Path) -> List[Type[BaseTool]]:
    """从模块路径发现所有工具类"""
    tools = []
    
    # 遍历包中的所有模块
    for _, modname, _ in pkgutil.iter_modules([str(package_path)]):
        if modname.startswith('_'):
            continue
            
        try:
            # 导入模块
            module = importlib.import_module(f"{module_path}.{modname}")
            
            # 查找BaseTool的子类
            for attr_name in dir(module):
                attr = getattr(module, attr_name)
                if (isinstance(attr, type) and 
                    issubclass(attr, BaseTool) and 
                    attr is not BaseTool and
                    hasattr(attr, 'name') and 
                    attr.name):
                    tools.append(attr)
                    
        except Exception as e:
            print(f"警告：加载模块 {modname} 时出错: {e}")
    
    return tools
```

### 8.6.2 加载流程

```python
def load_tools(data_provider=None) -> int:
```

## 8.7 单例模式与全局访问

### 8.7.1 单例实现

```python
# 单例实例
_tool_registry: Optional[ToolRegistry] = None
_tool_executor: Optional[ToolExecutor] = None

def get_tool_registry() -> ToolRegistry:
    """获取单例工具注册表"""
    global _tool_registry
    if _tool_registry is None:
        _tool_registry = ToolRegistry()
    return _tool_registry

def get_tool_executor() -> ToolExecutor:
    """获取单例工具执行器"""
    global _tool_executor
    registry = get_tool_registry()
    if _tool_executor is None:
        _tool_executor = ToolExecutor(registry)
    return _tool_executor
```

**工程化思维**：为什么使用单例？
- ✅ 统一管理所有工具

- ✅ 全局访问方便
