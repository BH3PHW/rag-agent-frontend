## 24.2 任务：添加"天气查询"工具

我们来添加一个新工具：查询天气

### 步骤1：创建工具定义

在 `chat-service/tools/tool_definitions/ 目录下创建新文件：

**weather_tool.py

```python
from typing import Dict, Any
from ..core.base import BaseTool

class WeatherQueryTool(BaseTool):
    """天气查询工具"""
    
    name = "query_weather"
    description = "查询指定城市的天气信息"
    
    parameters = {
        "type": "object",
        "properties": {
            "city": {
                "type": "string",
                "description": "城市名称，如北京、上海、深圳"
            }
        },
        "required": ["city"]
    }
    
    async def execute(self, arguments: Dict[str, Any]) -> Any:
        """
        执行天气查询
        
        注意：这里使用模拟数据，
        实际项目中应该调用真实的天气API
        """
        city = arguments["city"]
        
        # 模拟天气数据
        weather_data = {
            "北京": {
                "city": "北京",
                "temperature": "22°C",
                "condition": "晴",
                "humidity": "45%",
                "wind": "东南风 3级"
            },
            "上海": {
                "city": "上海",
                "temperature": "25°C",
                "condition": "多云",
                "humidity": "60%",
                "wind": "东风 2级"
            },
            "深圳": {
                "city": "深圳",
                "temperature": "28°C",
                "condition": "晴",
                "humidity": "70%",
                "wind": "南风 2级"
            }
        }
        
        # 返回对应城市的天气，或默认返回默认值
        return weather_data.get(
            city,
            {
                "city": city,
                "temperature": "N/A",
                "condition": "未知",
                "humidity": "N/A",
                "wind": "N/A"
            }
        )
```

### 步骤3：添加工具选择规则

在 `chat-service/agent_system.py` 中更新 `_rule_based_tool_selection` 方法：

```python
def _rule_based_tool_selection(self, query: str) -> Tuple[bool, List[Dict]]:
    tools_to_call = []
    
    # 原有规则...
    
    # 新增：天气查询规则
    if any(keyword in query for keyword in ["天气", "温度", "下雨", "晴天"]):
        # 简单提取城市名（简化版）
        city = self._extract_city(query)
        if city:
            tools_to_call.append({
                "name": "query_weather",
                "arguments": {"city": city}
            })
    
    return len(tools_to_call) > 0, tools_to_call

def _extract_city(self, query: str) -> Optional[str]:
    """
    从查询中提取城市名
    简化示例：
        "北京天气" -> 北京"
        "上海温度" -> "上海"
    """
    city_keywords = ["北京", "上海", "深圳", "广州", "杭州", "南京", "成都", "重庆"]
    
    for city in city_keywords:
        if city in query:
            return city
    
    return None
```

## 24.4 进阶：连接真实API

### 天气API集成示例

把模拟数据换成真实API：

```python
# weather_tool.py 更新版
import httpx
from typing import Dict, Any

class WeatherQueryTool(BaseTool):
    # ... 其他代码不变 ...
    
    async def execute(self, arguments: Dict[str, Any]) -> Any:
        city = arguments["city"]
        
        # 调用真实天气API（示例）
        api_key = "your_api_key"  # 替换为真实的Key
        url = f"https://api.weatherapi.com/v1/current.json?key={api_key}&q={city}"
        
        try:
            async with httpx.AsyncClient() as client:
                response = await client.get(url)
                data = response.json()
            
            # 解析API响应
            return {
                "city": city,
                "temperature": f"{data['current']['temp_c']}°C",
                "condition": data['current']['condition']['text'],
                "humidity": f"{data['current']['humidity']}%",
                "wind": data['current']['wind_dir']
            }
        except Exception as e:
            return {
                "error": str(e),
                "message": "天气查询失败，请稍后再试"
            }
```

## 24.5 工具开发最佳实践

### 检查清单

每添加一个工具，检查：

- ✅ 继承 BaseTool
- ✅ 定义 name, description, parameters
- ✅ 实现 execute() 方法
- ✅ 在 __init__.py 中导出
- ✅ 在 register_standard_tools() 中注册
- ✅ 在 agent_system.py 中添加规则
- ✅ 处理可能的异常
- ✅ 添加适当的错误处理

### 错误处理示例

```python
async def execute(self, arguments: Dict[str, Any]) -> Any:
    try:
        # 获取参数
        city = arguments["city"]
        
        # 验证参数
        if not city:
            return {"error": "城市名称不能为空"}
        
        # 执行操作
        result = await self._call_api(city)
        
        return result
    except KeyError as e:
        return {"error": f"缺少必要参数: {e}"}
    except httpx.HTTPError as e:
        return {"error": f"网络请求失败: {e}"}
    except Exception as e:
        return {"error": f"工具执行失败: {e}"}
```
